Privacy Policy for LifeOPS Focus Blocker

LifeOPS Focus Blocker stores the following information locally in the user’s browser using Chrome storage:
- the list of websites the user chooses to block
- the current Focus Mode on/off state

This extension does not collect personal information, browsing content, analytics data, or authentication data. It does not transmit user data to external servers. It does not sell, share, or transfer user data to third parties.

The extension uses Chrome’s declarativeNetRequest API only to apply user-selected site blocking rules and redirect blocked sites to an internal extension page while Focus Mode is active.
